Функции

NEXUS предоставляет множество функций для автоматизации задач. Вот полная документация по доступным функциям:

  • pit

    Описание: Помощник для работы с файлами.

    Команды:

    • pit copy source.txt destination.txt - Копирует файл.
    • pit move source.txt destination.txt - Перемещает файл.
    • pit delete file.txt - Удаляет файл.

  • nexus

    Описание: Помощник для управления проектом.

    Команды:

    • nexus init - Инициализирует новый проект.
    • nexus build - Собирает проект.
    • nexus deploy - Развертывает проект.

  • nex

    Описание: Язык программирования, предназначенный для компиляции в Python.

    Пример кода:

    import files

func greet(name) {
    say "Hello, " + name + "!"
}

greet("World")

set file "example.txt"
files create file
files write file "Hello, World!"
files read file as content
say content

  • Сайт проекта

    Описание: Официальный сайт с документацией и ресурсами.

    Ссылка: https://nexus-project.com
