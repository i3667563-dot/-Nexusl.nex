О проекте

NEXUS — это вспомогательная утилита, которая предоставляет мини-помощников для выполнения различных задач. Основная цель проекта — упростить и ускорить выполнение рутинных операций, предоставляя удобные инструменты и скрипты.

Функции

Основные компоненты NEXUS включают:

  • pit: Помощник для работы с файлами.
  • nexus: Помощник для управления проектом.
  • nex: Язык программирования, предназначенный для компиляции в Python.
  • Сайт проекта: Официальный сайт с документацией и ресурсами.
